# API-Super-Mario-Burguer-Front-end

#### Projeto em desenvolvimento, com o intuito de criar uma página de restaurante. No momeno é possível visualizar as rotas Home,Cardapio e Seja um fornecedor. 

![Capa Super Mario Burguer](https://user-images.githubusercontent.com/82067466/234884571-5f0f92f2-3fcc-4353-a552-2976af14fbac.jpg)

#### Clonando o repositório:
#### Tenha o Visual Studio Code instalado.
#### Vá até o repositório - Code - Clone e copie o link do repositório.
#### Abra o cmd - vá até a pasta que deseja fazer o clone - Insira o comando git clone + link do repositório.
#### Abra o projeto no Visual Studio Code - Use o comando npm install - Após baixar as dependências - use o comando npm run para abrir o projeto em seu navegador.
